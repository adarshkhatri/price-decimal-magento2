This extension will allow you to remove only `.00` from your frontend product prices.

Remove ".00" decimal from price in magento2

For Example,

$500.00 => $500

$19.99 => $19.99

<b>Removes from</b>:
<ol>
  <li>Catalog Page</li>
  <li>Product view page</li>
  <li>Compare page</li>
  <li>Cart page</li>
  <li>Checkout page</li>
  <li>Mention if you know other places ;)</li>
</ol>
